Simple WordPress plugin to hide the website from the public while it is being worked on or built.

== Description ==
This lightweight plugin makes it easy make your website hidden from public access using a page redirect that takes the user to a page of your choosing. 

You can choose from 3 modes: 
1. Online (default)
2. Coming Soon - for new websites that are under construction
3. Maintenance Mode - for existing websites undergoing changes

== Features ==
* Unique preview link - view the website while hidden from public
* Page selector - choose the page you want users to be redirected to or display default content
* Copyable bypass URL - copy the URL to the clipboard so you can easily share it with others to view the website while hidden from public
* 12 hour cookie expiration - the bypass URL will create a cookie on the device that expires after 12 hours. So long as the unique key remains unchanged, the link can be used repeatedly, each time setting a cookie with the 12 hour expiration. 
* Countdown timer - display a countdown timer to the date you want the website to be live
* Background images and customizable coming soon/maintenance mode page
* Preview coming soon/maintenance mode page


== Installation ==
1. Upload `simple-maintenance-mode` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings as needed

== Frequently Asked Questions ==
= How does it work? =
Once you activate the plugin, you can switch on the Coming Soon or Maintenance Mode, or the plugin settings by going to the WordPress Admin Settings menu and clicking on Maintenance Mode. 
You will be able to choose the mode you want to use from the settings page. The plugin will then redirect the user to the page you have chosen.

You can use a custom page for the coming soon/maintenance mode, or you can use the default page that the plugin creates and customize it from there. Add text, images, and other content to the page to customize it to your needs.

= How do I preview the coming soon/maintenance mode page? =
You can preview the coming soon/maintenance mode page by clicking the "Preview" button on the settings page. This will open a new tab in your browser so that you can see the page as the user would see it.

= Can I use my own custom page for the coming soon/maintenance mode? =
Yes, you can use your own custom page for the coming soon/maintenance mode. Simply select the page you want to use from the dropdown menu in the settings page.

= Can I use my own custom background image for the coming soon/maintenance mode? =
Yes, you can use your own custom background image for the coming soon/maintenance mode. Simply upload the image you want to use from the settings page.
